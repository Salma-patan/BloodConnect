# 🩸 BloodConnect - Blood Donation Management System

## 📖 Overview

BloodConnect is a web-based Blood Donation Management System developed using Django. The platform connects blood donors, recipients, hospitals, blood banks, and blood camps to simplify the process of finding and donating blood during emergencies.

The system provides secure authentication, blood request management, donor registration, blood bank information, blood camp updates, and administrative controls through a user-friendly interface.

## 🚀 Installation

### Clone Repository

git clone https://github.com/Salma-patan/BloodConnect.git

cd BloodConnect

### Create Virtual Environment

python -m venv venv

### Activate Virtual Environment

Windows:

venv\Scripts\activate

Linux/Mac:

source venv/bin/activate

### Install Dependencies

pip install -r requirements.txt

### Apply Migrations

python manage.py makemigrations

python manage.py migrate

### Run Development Server

python manage.py runserver

Open:

http://127.0.0.1:8000/
